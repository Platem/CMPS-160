Clicks and drawing

  • The program has an event that detects mouse clicks on the canvas.
  • It can check which mouse button was clicked. It only accepts right and left clicks.
  • Mouse clicks will draw a point. Theese points will be part of a polyline.

    • Left Click

    • Left click indicates a non-ending point.
    • It can be the first point of a polyline or an intermediate one.

    Right Click

    • Right click is used to indicate an ending point. This will be the last point of the current polyline and will draw the cylinder asociated to the line.
    • The exception for this is that if a right click is detected with no active polyline/cylinder it will work as the first point of it and won't be ending point.

    Line drawing

    • The program will draw the lines between every polyline points until the polyline is completed and the cylinder is drawn.

Mouse movement

  • The program will detect the mouse movement around the canvas.
  • If there is no active polyline nothing will happen. If there is any active polyline then a rubberband line will be drawn from the last point of the polyline to the current mouse position.

Cylinder drawing

  • When a polyline is completed a poly-cylinder will be asoicated to it. This means every point will become a node and between every consecutive pair of nodes a cylinder will be drawn.

    • Saving cylinders

    • Once a cylinder (or more) has been drawn the 'Save file' button can be pressed and our objects will be saved to a '.obj' file. Also the name of the file will be asked.
    • One the file is saved you can still draw more objects.

    Loading cylinders

    • Click on 'Choose file' to load a previously sabed '.obj' file containing objects. Then click 'Extract data' to draw them.
    • When a file is load the objects are saved in memory. This means that you can draw over the loaded file and continue the drawing.

    Normal Triggering

    • There's an option to toggle drawing of cylinders' faces normals.

    Ambient light

    • A blueish ambient light is added to the scene.

    Specular light

    • A greenish specular light can be toggled in the scene. This creates lightning highlights.
    • The glossiness of the cylinders surface can be adjusted to change the highlights intensity.

    Smooth shading

    • Smooth shading can be toggled to calculate colors for each vertex instead of each face (flat shading).

    Texture mapping

    • Texture mapping can be toggled for cylinders to draw with a texture blended with the color. 3 different textures provided.

Object interaction

    Picking Lights

    • Lights can be picked in the scene to toggle them. Left click on them to turn off/on.

    Picking Objects

    • Objects can be picked in the scene. Left click on them and they'll change.
    • Click in background to unpick.
    • Left click on another object to pick it (only one object picked at a time).

    Transforming Objects

    • Objects can be transformed. While an object is picked:
      • Left click and move to translate it (XY plane).
      • Middle click and move up/down to translate it (Z coord).
      • Right click and move to rotate it:
        • Horizontal movement: Rotate around X axis.
        • Vertical movement: Rotate around Z axis.

Camera

    Perspective view

    • Perspective projection view can be toggled to render the image this way instead of orthogonal projection.

    Panning

    • Click and drag in the background with no object picked will move the camera around. This can be achieved with 'WASD' keys.
    • Clicking Space key will reset the camera.

    Zooming (Perspective)

    • Scrolling the wheel with no object picked will change perspective camera fov. This can also be achieved using the slider in the options panel.

    Camera in/out movement.

    • Pressing middle button of the mouse (or 'O' key) will enter in camera movement mode.
    • Scrolling the wheel with this mode enabled will move the camera Z position.

Extra features

  • Here are some of the extra features added.

    • Slideshow

    • Two slideshows animations are available.
    • Look around (Perspective): Double click on an object to automatically center the camera on it and see a rotation animation looking around the object.
    • Examine: pick an object
    • Pressing ESC key will get out of slideshow mode.

    Surface Triggering

    • There's an option to toggle drawing of cylinders' surfaces.

    Point Triggering

    • There's an option to toggle drawing of cylinders' points.

    Skeleton Triggering

    • There's an option to toggle drawing of cylinders' skeleton.

    Surface Triggering

    • There's an option to toggle drawing of cylinders' surface.

    Color selection

    • Color selection is enabled for all drawable elements:
      • Ambient light
      • Diffuse light
      • Specular light
      • Face normals
      • Cylinders skeleton
      • Cylinders points

    Camera Rotation

    • Four buttons have been added to rotate objects in the viewport. This is done using a rotation matrix around the Y/X axis.
    • This rotation can be achieved using the arrow keys too.
    • A 'center' button has been added to center the object again (Enter key).

    Perspective adjustments

    • Options to adjust perspective parameters have been added.

    Multiple object saving / loading

    • I've modified the ioSOR library to make it posible to save several objects in one file (and be able to load a file containing more than one object).
    • This modified 'mod_ioSOR.js' file is located in the 'lib' folder.

    Object management

    • In the right side of the page theres a list with current objects in canvas.
    • There are two options for each object: toggle visibility and remove them.

    Light sources management

    • In the right side of the page theres a list with current light sources in canvas.
    • More sources can be added and configured.
